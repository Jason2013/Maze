“迷宫探险”游戏说明：

迷宫的入口在左上角、出口在右下角。迷宫初始大小为宽8高6，你可以
用调整迷宫的大小。迷宫宽度范围是2到48，高度范围是2到36。


操作说明：

HOME      减小迷宫的宽度
END       增大迷宫的宽度
PAGEUP    减小迷宫的高度
PAGEDOWN  增大迷宫的高度
F2        按照当前宽、高生成一个新迷宫
F1        显示正确路线
F3        重玩当前迷宫
LEFT/UP/RIGHT/DOWN 在迷宫中向左、上、右、下走一步。走过的路线显
          示为黄色，回头路显示为红色。
ESC       退出游戏

源程序编译连接说明：

在Turbo C++ 3.0 “Project”菜单中选“Open Project...”菜单项，
弹出“Open Project File”，打开MazeCn.Prj项目文件。

然后在“Compile”菜单中选“Build All”菜单项，编译连接后，生成
可执行程序 。

最后在“Run”菜单中选“Run”菜单项，运行程序。
